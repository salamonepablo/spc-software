# Session: Preserve Customer IDs from Access to SQL Server

**Last Updated:** 2026-03-07

## Goal

Preservar los IDs originales de clientes de Access en SQL Server para que los clientes existentes puedan seguir usando sus códigos del sistema legacy. El cliente 370 "Castelucci, Luis Octavio" debe mantener su ID en la nueva base de datos.

---

## Instructions

- Exportar TODAS las tablas desde Access a CSV
- Limpiar completamente la base de datos SQL Server
- Reimportar todo desde CSV preservando los IDs originales
- El filtro de búsqueda de clientes debe buscar por: Código (Id), Razón Social, Nombre Fantasía
- Mostrar columna "Codigo" (Id) en el CRUD de clientes
- Resolver error de compilación: `BulkConfig` no tiene propiedad `KeepIdentity` en EFCore.BulkExtensions 10.0.1

---

## Discoveries

1. **KeepIdentity no existe** en `BulkConfig` de EFCore.BulkExtensions 10.0.1
   - La propiedad `KeepIdentity` no existe en esta versión
   - Necesitamos usar SQL directo con `IDENTITY_INSERT ON` y `SqlBulkCopy`

2. **Las facturas tienen IdSucursal = 0, 2, 5**
   - Solo existían sucursales con ID 2 y 5
   - Se agregó una sucursal por defecto con ID=1 para los casos donde IdSucursal=0

3. **Verificación exitosa**: Cliente 370 = "CASTELUCCI LUIS OCTAVIO" (CERRAJERIA CASTELMARI)

---

## Accomplished

### Fix KeepIdentity Error (COMPLETADO)

Creado método `BulkInsertWithIdentityAsync<T>()` que usa:
- `SqlConnection` directa con transacción
- `SET IDENTITY_INSERT [tabla] ON` antes de insertar
- `SqlBulkCopy` con `SqlBulkCopyOptions.KeepIdentity` para inserción rápida
- `SET IDENTITY_INSERT [tabla] OFF` después de insertar

Archivos modificados:
- `SPC.Migration/ImportFromCsv.cs` - Nuevo método helper + uso en todas las importaciones
- `SPC.Migration/Program.cs` - Nuevo método helper para migración directa desde Access

### Limpieza de Base de Datos (COMPLETADO)

Agregada opción `--clean` al programa de migración:
- Deshabilita restricciones FK con `sp_MSforeachtable`
- Elimina datos en orden correcto (respetando dependencias)
- Resetea seeds de identidad
- Rehbilita restricciones FK
